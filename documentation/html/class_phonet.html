Detailed Description

A Phonet is a vector of phonemes responsible for creating the phonetic version of a word. Phonets should only be used to store the phonetic spellings of words and not there actual spellings.

Operations include:
  • Constructing an empty Phonet
  • Constructing a randomly generated Phonet
  • Constructing an already existing Phonet
  • Generating a random Phonet
  • Printing a Phonet to console
